HTML Style and Semantics Tags

HTML provides style and semantics tags to structure and style content effectively.

<section>

Defines a thematic grouping of content, often with a heading.

<article>

Represents a self-contained composition within a document, such as a blog post.

<header>

Represents introductory content at the beginning of a section or a page.

<footer>

Represents a footer for a section or the whole document.

<nav>

Defines navigation links for the document.

The <mark> element highlights text for reference purposes.

The <strong> element indicates strong importance.

The <em> element emphasizes text.

The <abbr> element defines an abbreviation or acronym.

The <code> element is used for displaying code snippets.

The <kbd> element represents user input or keyboard input.

The <samp> element is used for sample output from a program.

The <var> element represents a variable.

<blockquote>

Represents a section that is a quotation from another source.

<q>

Represents a short inline quotation.

<address>

Represents the contact information for the author or owner of a document.

<dfn>

Represents the defining instance of a term.

<time>

Represents a specific period in time or a date.

The <progress> element represents the completion progress of a task.

The <meter> element represents a scalar measurement within a known range.
